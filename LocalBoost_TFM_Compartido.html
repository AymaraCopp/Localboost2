Firebase Real-Time Sync — Setup Guide
One-time setup · Free · Takes about 10 minutes · No coding required
Firebase is Google's free database service. Once set up, any change one teammate makes — a note, a task, a resource — instantly appears on everyone else's screen. The free plan is more than enough for this project.
1
Create a free Firebase project
  1. Open console.firebase.google.com in your browser and sign in with any Google account.
  2. Click "Add project" (or "Create a project" if it's your first time).
  3. Type a project name — for example localboost-tfm — then click Continue.
  4. On the "Google Analytics" screen, you can turn it off (it's not needed). Click Create project.
  5. Wait about 15 seconds for the project to be created, then click Continue.
💡 Any Gmail account works — it doesn't have to be your school email.
2
Enable the Realtime Database
  1. In the left sidebar of the Firebase console, click "Build" to expand the menu, then click "Realtime Database".
  2. Click the blue "Create Database" button.
  3. A dialog appears asking for a location — choose any region (e.g. United States) and click Next.
  4. On the security rules screen, select "Start in test mode". This allows your team to read and write data for 30 days. Click Enable.
  5. Your database is now live. You'll see a URL at the top that looks like:
    https://localboost-tfm-default-rtdb.firebaseio.com/
    Copy and save this URL — you'll need it in Step 4.
💡 "Test mode" is fine for a private academic project. If you want extra security after 30 days, you can extend the rules — just search "Firebase rules extend test mode".
3
Get your API Key and Project ID
  1. Click the ⚙️ gear icon next to "Project Overview" in the top-left sidebar, then click "Project settings".
  2. On the "General" tab, scroll down to the section titled "Your apps".
  3. If you don't see an app listed, click the </> Web icon to register a web app. Give it any nickname (e.g. tfm-app) and click Register app. You don't need to install anything — just click Continue to console.
  4. Now scroll down to the firebaseConfig code block. It will look something like this:
firebaseConfig (example — yours will have different values)
const firebaseConfig = {
  apiKey: "AIzaSyB...your-key-here...",
  authDomain: "localboost-tfm.firebaseapp.com",
  databaseURL: "https://localboost-tfm-default-rtdb.firebaseio.com",
  projectId: "localboost-tfm",
  storageBucket: "localboost-tfm.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};
📋 You only need 3 values: apiKey, databaseURL, and projectId. Copy them somewhere (a sticky note, a text file).
4
Paste your credentials into this HTML file
  1. Open the file LocalBoost_TFM_Compartido.html with a plain text editor:
      • Windows: Right-click the file → "Open with" → Notepad
      • Mac: Right-click → "Open with" → TextEdit (make sure it's in plain text mode: Format → Make Plain Text)
      • Any OS: VS Code (free) works great too
  2. Use Ctrl+F (or Cmd+F on Mac) to search for: YOUR_API_KEY_HERE
  3. Find these 3 lines near the top of the script section and replace the placeholder values with your real ones:
Find these lines and replace the placeholder text:
const FIREBASE_API_KEY    = "AIzaSyAjKV4u354V2pzfzNqaPrQ0wqpR-tn5_p8";
const FIREBASE_DB_URL     = "https://localboost-2-default-rtdb.europe-west1.firebasedatabase.app",
  projectId: "localboost-2";
const FIREBASE_PROJECT_ID = "localboost-2";
After editing, it should look like this (with your real values):
const FIREBASE_API_KEY    = "AIzaSyB...your-real-key...";
const FIREBASE_DB_URL     = "https://localboost-tfm-default-rtdb.firebaseio.com";
const FIREBASE_PROJECT_ID = "localboost-tfm";
  1. Save the file (Ctrl+S or Cmd+S).
  2. Open it in your browser to test — the yellow setup banner on this page should disappear, and the green 🟢 sync indicator should appear in the top bar.
⚠️ Make sure to keep the quotes " " around the values. Do not add spaces or extra characters.
5
Share the updated file with your team via Google Drive
  1. Upload the updated LocalBoost_TFM_Compartido.html to your shared Google Drive folder.
  2. Share the folder with both teammates — right-click the folder → "Share" → add their email addresses → set permission to "Viewer" (they can download it).
  3. Each teammate downloads the file and opens it in their browser (Chrome or Firefox recommended).
  4. On first open, each person enters their name and picks their team color on the sign-in screen.
  5. That's it — all three of you are now synced in real time! 🎉
🔄 If someone edits a note or adds a task, the others will see it update within a few seconds — no page refresh needed. The green dot 🟢 in the top bar confirms you're live.
🆓 100% Free — Firebase Spark plan is free forever, with limits far beyond what this project needs.
🔒 Private — only people with the HTML file and your Firebase credentials can access the data.
📴 Works offline too — if someone opens the file without internet, their data saves locally and syncs automatically when they reconnect.
